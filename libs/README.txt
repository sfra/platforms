Put socket.io.js into this folder. Just type:
npm install socket.io
in terminal, and copy the file from appropriate location.
Moreover, download from http://requirejs.org/docs/download.html file
require.js and put it to the current folder.
Moreover when you installed socket.io, copy the client to to current folder:
```
cp ~/node_modules/socket.io/node_modules/socket.io-client/dist/socket.io.js .

```
